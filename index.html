📖 使い方

🎤 音声入力

  • マイクボタンを押すと録音開始。もう一度押すと停止。
  • 話した言葉がリアルタイムでテキストエリアに表示されます。
  • ネット接続が必要です（Google の音声認識を使用）。

⌨️ ショートカットキー

Ctrl + M音声入力の開始 / 終了
Ctrl + Cテキストを全文コピー
Ctrl + Aテキストを全選択
Ctrl + Vテキストエリアにペースト
Ctrl + Z元に戻す

※ テキストエリアにフォーカスがある場合は通常のブラウザ動作と同じです。

📋 コピーして使う

  • 「コピー」ボタンまたは Ctrl+C でテキストをコピー。
  • Google ドキュメント・フォームなど好きなアプリに貼り付けて使えます。

🔤 ふりがな表示

  • 上部の学年ボタンで学年を選ぶと、その学年でまだ習っていない漢字にふりがなを付けて下に表示します。
  • ふりがなはプレビュー表示のみで、コピーされるテキストには含まれません。

📝 履歴

  • 「消す／消去」ボタンを押すと、テキストが自動的に履歴に保存されます。
  • 履歴のカードをクリックすると、そのテキストを呼び出せます。
  • ✕ ボタンで履歴を1件ずつ削除できます。
  • 最新5件まで保存。ページを閉じても残ります。
